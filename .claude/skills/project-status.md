# プロジェクト状況確認 スキル

ダッシュボード情報・タスク・マイルストーン・レビューを横断的に取得し、プロジェクトの全体像をレポートします。

## ワークフロー

### 1. ダッシュボード取得
MCP `dashboard_get` でタスク統計を取得。

### 2. マイルストーン確認
MCP `milestone_list` でマイルストーン一覧と期限を取得。

### 3. ボール状態確認
MCP `ball_query` (ball=client) と (ball=internal) でそれぞれのタスクを取得。

### 4. オープンレビュー確認
MCP `review_list` (status=open) で未完了レビューを取得。

### 5. レポート出力
以下の形式でサマリーを出力：

```
## プロジェクト状況サマリー

### タスク統計
- 全タスク: XX件
- クライアント待ち: XX件 / 社内作業中: XX件
- 進行中: XX件 / レビュー中: XX件 / 完了: XX件

### マイルストーン
| 名前 | 期限 | 状態 |
|------|------|------|
| ... | YYYY-MM-DD | 進行中/完了 |

### ブロッカー
- レビュー待ち: XX件
- クライアント待ち: XX件（最も古い: X日前）
```

## 使用するMCPツール
| 目的 | ツール |
|------|--------|
| 統計 | `dashboard_get` |
| MS一覧 | `milestone_list` |
| ボール状態 | `ball_query` |
| レビュー | `review_list` |
| タスク詳細 | `task_list` |
