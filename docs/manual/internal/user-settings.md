# ユーザー・組織設定

アカウント情報、表示設定、組織管理、メンバー管理、課金、外部連携、通知設定など、個人と組織に関する設定を行うページ群です。

---

## アクセス方法

左ナビゲーション下部のユーザーアイコンまたは設定メニューから各設定ページに遷移します。

---

## アカウント設定

URL: `/settings/account`

プロフィール情報の編集を行います。

| 項目 | 内容 | 編集 |
|------|------|------|
| **表示名** | タスクの担当者表示などで使われる名前 | テキスト入力（最大50文字） |
| **メールアドレス** | 認証に使用しているメールアドレス | 変更不可（読み取り専用） |
| **アバター** | プロフィール画像 | 準備中 |

画面下部にユーザー ID・アカウント作成日・最終ログイン日が表示されます。

> **注意**: 表示名の変更は「保存」ボタンを押すまで反映されません。

---

## 環境設定

URL: `/settings/preferences`

表示や UI のカスタマイズを行います。設定はすべて **自動保存** されます（保存ボタン不要）。

### テーマ

| 選択肢 | 説明 |
|--------|------|
| Light | ライトテーマ |
| Dark | ダークテーマ（準備中） |
| System | OS設定に連動（準備中） |

> **注意**: Dark / System は設定の保存は可能ですが、テーマの切替表示はまだ反映されません。

### 言語

| 選択肢 | 説明 |
|--------|------|
| 日本語 | デフォルト |
| English | 準備中（設定は保存されます） |

### タイムゾーン

以下の8つから選択できます。

| タイムゾーン | 略称 |
|-------------|------|
| Asia/Tokyo | JST (UTC+9) |
| America/New_York | EST (UTC-5) |
| America/Los_Angeles | PST (UTC-8) |
| Europe/London | GMT (UTC+0) |
| Europe/Berlin | CET (UTC+1) |
| Asia/Shanghai | CST (UTC+8) |
| Asia/Seoul | KST (UTC+9) |
| Australia/Sydney | AEST (UTC+10) |

### サイドバーデフォルト状態

- 「デフォルトで折りたたむ」をオンにすると、サイドバーを折りたたんだ状態で開始します

## 組織設定

URL: `/settings/organization`

組織の基本情報を管理します。

| 項目 | 内容 | 編集権限 |
|------|------|---------|
| **組織名** | 組織の表示名（最大100文字） | オーナーのみ |
| **組織 ID** | システム内部の識別子 | 変更不可 |
| **あなたの役割** | Owner / Member / Client | 変更不可（表示のみ） |

> **注意**: 組織名の変更はオーナーのみ可能です。メンバーやクライアントは閲覧のみできます。

---

## メンバー管理

URL: `/settings/members`

組織のメンバー一覧と管理を行います。

### メンバー一覧

各メンバーの以下の情報が表示されます。

- アバター（または頭文字アイコン）
- 表示名（自分の場合は「あなた」バッジ付き）
- メールアドレス
- 参加日
- 役割

### メンバー操作（オーナーのみ）

| 操作 | 説明 |
|------|------|
| **役割変更** | ドロップダウンで Owner / Member / Client を切り替え |
| **削除** | 確認ダイアログ後にメンバーを削除 |
| **招待** | メールアドレスと役割（Member / Client）を指定して招待 |

> **注意**: オーナー以外のユーザーはメンバー一覧の閲覧のみ可能です。

## 外部連携（個人）

URL: `/settings/integrations`

個人の外部サービス接続を管理します。

### Google カレンダー

- OAuth認証でGoogleアカウントを接続
- 接続後: アカウントメール・スコープ・最終同期日が表示
- 日程調整で FreeBusy（空き時間）を参照できるようになります

### ビデオ会議アカウント

| プロバイダー | 説明 |
|-------------|------|
| **Google Meet** | Google カレンダー連携から自動利用可能 |
| **Zoom** | 別途接続が必要 |
| **Microsoft Teams** | 別途接続が必要 |

---

## 組織の外部連携

URL: `/settings/org-integrations`

組織全体の外部サービス設定を管理します（オーナーのみ変更可能）。

### GitHub 連携

- 組織のGitHubアカウントを接続
- 接続後、各プロジェクト設定でリポジトリを紐付け可能

### Slack 連携

- Slackワークスペースを接続（OAuth または手動トークン入力）
- 接続後、各プロジェクト設定で通知先チャンネルを選択可能
- 手動入力の場合、Bot Token は `xoxb-` で始まる文字列

### AI 設定

LLM API キーを登録して、Slack での AI メンション機能を有効にします。

| 項目 | 選択肢 |
|------|--------|
| **プロバイダー** | OpenAI / Anthropic |
| **モデル** | OpenAI: gpt-4o-mini, gpt-4o / Anthropic: claude-sonnet-4-5, claude-haiku-4-5 |
| **API キー** | 暗号化して保存。プレフィックスのみ表示 |

> **注意**: API キーの検証: OpenAI は `sk-` で始まる文字列、Anthropic は `sk-ant-` で始まる文字列が必要です。

## API キー管理

URL: `/settings/api-keys`

外部ツール連携用の API キーを生成・管理します。

### キーの作成

1. キー名を入力（例: 「Claude Code用」）
2. 許可するプロジェクトを選択（1つ以上必須）
3. 許可するアクション（read は必須、write / delete はオプション）を選択
4. 「作成」をクリック

> **注意**: 生成された API キーは **一度しか表示されません**。安全な場所に保存してください。

### 発行済みキー一覧

各キーについて以下が表示されます。

- キー名・キープレフィックス（`tsk_XXXXXX...`）
- 紐付きプロジェクト名
- 許可されたアクション
- 作成日・最終使用日

## プランと請求

URL: `/settings/billing`

使用状況とサブスクリプションの管理を行います。

### 使用状況

現在のプランの利用枠と使用量が表示されます。

### プランアップグレード

| プラン | 上限 |
|--------|------|
| Free | プロジェクト5件・メンバー5人・クライアント5人 |
| Pro | プロジェクト20件・メンバー20人・クライアント20人 |
| Enterprise | 無制限 |

有料プラン契約後は Stripe ポータルからプラン変更・支払い方法の更新・請求書ダウンロードが可能です。

## 通知設定

URL: `/settings/notifications`

メール通知の設定を管理します。

> **注意**: メール通知機能は準備中です。設定は保存されますが、現時点ではメール配信は行われません。

### マスタートグル

「メール通知」のトグルで、すべてのメール通知をまとめて有効/無効にできます。

### 個別通知タイプ

| 通知タイプ | トリガー |
|-----------|---------|
| タスク割り当て | タスクがあなたに割り当てられた時 |
| メンション | コメントであなたがメンションされた時 |
| 承認依頼 | 承認を依頼された時 |
| クライアント応答 | クライアントが確認・回答した時 |
| 会議リマインダー | 予定された会議の前日 |

### ダイジェストメール

未読通知のまとめメールの頻度を選択できます。

| 選択肢 | 説明 |
|--------|------|
| なし | ダイジェストメールを送らない |
| 毎日 | 毎日まとめを送信 |
| 毎週 | 毎週まとめを送信 |
