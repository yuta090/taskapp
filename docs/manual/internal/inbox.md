# 受信トレイ

アプリ内通知の一覧確認と対応を行うページです。通知の既読管理、タイプ別のアクション実行、前後ナビゲーションなどの機能を備えています。

---

## アクセス方法

左ナビゲーション上部の「受信トレイ」をクリックします。未読件数がバッジで表示されます。

URL: `/inbox`

---

## 画面構成

3ペインレイアウトの中央に通知一覧、右側にインスペクターが表示されます。

### ヘッダー

| 要素 | 説明 |
|------|------|
| タイトル | 「受信トレイ」＋ 未読件数バッジ（青） |
| キーボードヒント | `↑↓ 移動` ・ `Enter 詳細へ` |
| 一括既読ボタン | 「すべて既読」（未読がある場合のみ表示） |

---

## 通知一覧

### 各行の表示要素

| 要素 | 説明 |
|------|------|
| アイコン | 通知タイプを示すアイコン（色付き） |
| タイトル | 未読は太字、既読はグレー |
| メッセージ | 1行に省略表示 |
| バッジ | 「緊急」（赤）/「要対応」（アンバー）/「対応済み」（グレー） |
| 未読インジケーター | 青いドット（未読のみ） |
| メタ情報 | 経過時間・送信者名・プロジェクト名 |

### 左端ボーダーの色

| 色 | 条件 |
|----|------|
| 赤 | 緊急 かつ 未読 |
| 青 | 現在選択中 |
| なし | それ以外 |

## 既読・対応済みの仕組み

通知には2段階の状態があります。

| 状態 | 条件 | 見た目 |
|------|------|--------|
| **未読** | `read_at` が未設定 | 太字 + 青ドット |
| **既読・未対応** | `read_at` あり、`actioned_at` なし | 通常表示 + 「要対応」バッジ |
| **対応済み** | `actioned_at` あり | グレー表示 + 「対応済み」バッジ |

- 通知をクリックすると自動的に **既読** になります
- アクション（承認、対応開始など）を実行すると **対応済み** になります
- 対応済みにすると既読も同時に設定されます

> **注意**: 「要対応」バッジが表示される通知タイプは、承認依頼・タスク割り当て・ボール移動・確認依頼・緊急確認・仕様決定依頼です。

## インスペクター（右パネル）

通知を選択すると右側にインスペクターが表示されます。

### 表示内容

| セクション | 説明 |
|-----------|------|
| ヘッダー | 通知タイプ名・緊急バッジ・前後ナビボタン・閉じるボタン |
| アイコン・タイトル | 大きいアイコン + タイトル + 日時・送信者名 |
| メッセージ | 通知本文 |
| コメント | 引用表示（グレーボックス） |
| 質問 | 強調表示（アンバーボックス） |
| 関連タスク | タスク名・説明（4行まで）・ステータスドロップダウン・期限 |
| アクションパネル | 通知タイプ別の操作ボタン（後述） |
| 会議サマリー | 件名・統計（決定数/未決数/要対応数）・本文 |
| 会議情報 | 会議名・日時（緑ボックス） |

### タイプ別アクションパネル

#### 承認依頼

- 「承認する」ボタン（緑）→ 承認処理を実行
- 「差し戻す」ボタン（赤枠）→ 理由入力テキストエリアが表示 → 理由を入力して送信

#### タスク割り当て / ボール移動

- 「対応開始して次へ」ボタン（青）→ タスクステータスを「進行中」に変更

#### 確認依頼 / 緊急確認

- 「日程を回答する」ボタン（青）→ 日程調整の回答ページにリンク

#### 仕様決定依頼

- 「決定済みにする」ボタン（青）→ 仕様タスクの `decision_state` を `decided` に更新

### フッターボタン

| ボタン | 説明 |
|--------|------|
| 「完了にして次へ」 | 対応済みにして次の通知に自動移動（600ms後） |
| 「詳細を見る」 | タスクやミーティングの詳細ページに遷移 |
| 「既読にして次へ」/「次の通知へ」 | 既読にして次の通知に移動 |

---

## キーボード操作

| キー | 操作 |
|------|------|
| `↑` または `k` | 前の通知に移動 |
| `↓` または `j` | 次の通知に移動 |
| `Escape` | インスペクターを閉じる |
| `Enter` | 通知の詳細リンクを開く |

---

## ベルアイコン（未読バッジ）

左ナビゲーションの「受信トレイ」項目に未読件数が表示されます。

- サイドバー展開時: 数字バッジ
- サイドバー折りたたみ時: 小さなドット
- 30秒ごとに自動更新されます
