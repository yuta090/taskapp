# ダッシュボード

プロジェクトの全体状況を一目で確認できるページです。タスクの進捗、リスク、期限、クライアント対応状況などを集約表示します。

---

## アクセス方法

左ナビゲーションのプロジェクト名をクリックし、「ダッシュボード」を選択します。

URL: `/:orgId/project/:spaceId/dashboard`

---

## 画面構成

ダッシュボードは以下のセクションで構成されています。

### KPI カード（4枚）

画面上部に 4 つの指標カードが並びます。

| カード | 表示内容 | 補足 |
|--------|---------|------|
| **残タスク** | 完了以外のタスク数 | サブテキストに完了数を表示 |
| **ボール（社内/クライアント）** | `社内数 / クライアント数` 形式 | アクティブタスクの内訳 |
| **期限超過** | 期限を過ぎたアクティブタスク数 | 1件以上で赤色表示 |
| **レビュー待ち** | オープン中のレビュー数 | 1件以上でアンバー色表示 |

---

### クライアント確認が必要

クライアントにボールがあるタスクのうち、フォローアップが必要なものを表示します。

#### 要フォロー（赤）

以下のいずれかに該当するタスクです。

- 期限超過（`due_date` が過去）
- ボールを渡して **7日以上** 経過 かつ 期限まで3日以内

#### そろそろ確認（アンバー）

以下のいずれかに該当するタスクです。

- ボールを渡して **5日以上** 経過
- 期限まで7日以内

各行にはタスク名・期限までの日数・渡してからの経過日数が表示されます。クリックするとタスク詳細に遷移します。

> **Tips**: 「クライアント確認が必要」セクションが空の場合は「現在フォローが必要なタスクはありません」と表示されます。

---

### マイルストーン進捗

アクティブなマイルストーンごとに以下を表示します。

| 項目 | 説明 |
|------|------|
| マイルストーン名 | クリック不可（表示のみ） |
| 期限 | 期限までの残日数 |
| リスクバッジ | 高リスク（赤）・中リスク（アンバー）・低リスク（緑）・完了（グレー） |
| プログレスバー | 完了タスク数 / 全タスク数 の割合 |

#### リスク判定ロジック

リスクレベルは直近14日間のタスク完了ペース（ベロシティ）と残タスク数から自動計算されます。

| 条件 | リスクレベル |
|------|-------------|
| 全タスク完了 | 完了 |
| 期限超過 | 高リスク |
| 14日間の完了タスクが 0 件 | 高リスク |
| 必要日数 / 残日数 > 1.5 | 高リスク |
| 必要日数 / 残日数 > 1.0 | 中リスク |
| 上記以外 | 低リスク |

---

### ボール所在

アクティブタスクのボール分布を棒グラフで表示します。

- **社内**（インディゴ）: 社内にボールがあるタスク数
- **クライアント**（アンバー）: クライアントにボールがあるタスク数

---

### 期限が近いタスク

直近 **7日以内** に期限を迎えるアクティブタスクを最大8件表示します。

- 期限超過: 赤色のドット
- 期限まで2日以内: アンバー色のドット
- それ以外: 緑色のドット

各行をクリックするとタスク詳細に遷移します。

---

### 直近の予定

今後予定されているミーティングを最大5件表示します。

- 日付（M/D形式）と時刻（HH:MM形式）
- ミーティング名

各行をクリックするとミーティング詳細に遷移します。

---

## 操作について

ダッシュボードは **読み取り専用** です。データの編集はタスク一覧やミーティング一覧から行ってください。
