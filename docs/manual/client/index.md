## かんたん操作ガイド

| やりたいこと | 操作方法 |
|-------------|---------|
| 対応が必要なタスクを確認 | ダッシュボードの黄色バッジ付きタスクを確認 |
| タスクにコメントする | タスクをクリック → コメント欄に入力 |
| 日程調整に回答する | 会議ページ → 候補日時に回答 |
| レビューを承認する | ダッシュボード「対応が必要」→ 承認ボタン |
| 決定事項を確認する | タスク詳細 → 「確認済み」ボタン |

## よくある質問（FAQ）

**Q: ポータルにアクセスできなくなりました**
A: 招待メールのリンクの有効期限が切れている可能性があります。開発チームに再招待をご依頼ください。

**Q: 黄色のバッジは何ですか？**
A: あなたの確認・回答が必要なタスクを示しています。優先的にご対応ください。

**Q: タスクのステータスを変更できますか？**
A: いいえ。ステータスの変更は開発チームが行います。ご要望がある場合はコメントでお伝えください。

**Q: 間違えて承認してしまいました**
A: 開発チームにコメントでお知らせください。対応いたします。

**Q: 複数の依頼が来ていますが、優先順位は？**
A: ダッシュボードでは期限が近いものから順に表示されます。上から順にご対応いただくのがおすすめです。
