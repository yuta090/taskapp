## 2. Problem — 「本業じゃないこと、多すぎませんか？」

### セクションタイトル
```
Problem

プロジェクトに関わる全員が、
「本業じゃないこと」に追われている。
```

### ペルソナ別の痛み（カード形式、2x2グリッド）

**開発者の場合:**
- 「管理画面を開いて、ステータス変えて、コメント書いて...1日30分、コード書く時間が削られる」
- 吹き出し: 「Backlogの更新忘れてませんか？」
- 定量: エンジニアの管理業務時間 = 平均1日30分〜1時間

**PMの場合:**
- 「エンジニアの進捗をヒアリング → Excelに転記 → クライアントに報告。私は転記ロボットか？」
- 吹き出し: 「週次報告書、今週もですか...」
- 定量: PMの週次報告作成 = 平均2〜3時間/週

**クライアントの場合:**
- 「今どこまで進んでるの？聞かないとわからない。承認した記録もどこかにいった」
- 吹き出し: 「あの件、どうなりました？」
- 定量: 進捗確認のチャット = 1案件あたり週3〜5回

**代理店の場合:**
- 「制作会社の見積もりをExcelに転記、マージン計算して、クライアント用に作り直す。全部手作業」
- 吹き出し: 「原価率、合ってるよね...？」
- 定量: 月末の請求処理 = 案件ごとに平均2時間

## 5. Feature: Portal — 「URLを送るだけ」

### セクションタイトル（id="for-clients"）
```
For Clients

進捗が見える。だから聞かなくていい。
```

### 本文
```
クライアントにはポータルのURLを共有するだけ。
アカウント登録もアプリのインストールも不要です。
プロジェクトの進捗、承認待ちのアクション、スケジュールが一目で確認できます。
```

### 特徴リスト
- **アカウント登録不要**: URLを開くだけ。ITリテラシーを問いません
- **進捗が一目瞭然**: マイルストーン・タスクの完了率がリアルタイム表示
- **承認はワンクリック**: 見積もり・デザイン・仕様の承認がポータル上で完結
- **証跡が残る**: 誰が・いつ・何を承認したか全て記録。監査対応にも

### セキュリティ・権限（発注者の不安を解消）
```
安心して使えるセキュリティ:
- ポータルURLは期限付き・個別発行。第三者がアクセスできません
- 閲覧範囲はプロジェクト管理者が設定（公開するセクションを選択可能）
- 閲覧・承認の履歴は全て記録され、CSVでエクスポート可能
- 通信はTLS 1.3暗号化。データはAES-256で保存
```

### ビジュアル
既存FeaturePortalのUIモック
+ 実際のポータル画面のスクリーンショット（モックとの対比で信頼感）

### このペルソナ向けCTA

## 7. For Agency — 「3者の間をシンプルに」（独立セクション化）

### セクションタイトル（id="for-agencies"）
```
For Agencies

クライアントと制作会社の"間"を、一画面で管理。
```

### 本文
```
代理店モードを有効にすると、原価・マージン・売値の3層管理が可能に。
制作会社は原価を入力。代理店はマージンを設定。クライアントには売値だけが表示されます。
Excelでの転記・計算ミスから解放されます。
```

### 具体的な管理フロー（図）
```
[制作会社]                   [代理店]                    [クライアント]
 工数: 16h                    マージン率: 35%              表示される金額:
 単価: ¥8,000/h        →     原価: ¥128,000        →     ¥172,800
 原価: ¥128,000               売値: ¥172,800               (原価は非表示)

 進捗: 60%完了          →     そのまま転送            →     60%完了
 (制作会社が更新)              (転記不要)                   (ポータルで確認)
```

### 特徴リスト
- **原価/マージン/売値の一元管理**: 案件ごとの粗利がリアルタイムで把握
- **制作会社の進捗をそのまま共有**: 手動転記不要。見せたい情報だけ選択可能
- **ロール別の権限管理**: 制作会社に見えるもの・クライアントに見えるものを個別設定
- **案件横断の粗利ダッシュボード**: 全案件の原価率・粗利率を一覧表示
- **会計ソフト連携**: CSV出力でfreee等との連携に対応

### 信頼データ
```
導入前: 月末の請求処理 = 案件あたり平均2時間 × 10案件 = 20時間
導入後: 案件あたり15分 × 10案件 = 2.5時間
→ 月17.5時間の削減（約87%減）
```

### ビジュアル
3者関係の図（制作会社 → 代理店 → クライアント）
+ 代理店ダッシュボードの実画面スクリーンショット

### このペルソナ向けCTA
```
[代理店モードを試す →]  Businessプランに含まれています（追加料金なし）
```

### セクションタイトル
```
Getting Started

5分で、プロのPM環境が整う。
```

### ステップ（アニメーション付き）
```
Step 1: アカウント作成
  → メールアドレスだけで即開始。Google/GitHubアカウントでもOK

Step 2: テンプレートから選択
  → 受託開発、Web制作、アプリ開発、SaaS開発など業種別テンプレート
  → マイルストーン・タスク構成が最初から入っている

Step 3: 既存データの移行（オプション）
  → BacklogのCSV、GitHub Issues、スプレッドシートのコピペ
  → AIに渡すだけでタスクが自動登録
  → 移行チェックリスト付き（履歴・添付ファイルの扱いを事前確認）

Step 4: チームを招待 & ポータルURLを共有
  → 開発メンバーはメール招待
  → クライアントにはポータルURLを送るだけ
```

### 移行の安心情報（PMの不安を解消）
```
Backlogからの移行ガイド:
✓ タスク・マイルストーン → CSVインポートで自動登録
✓ コメント履歴 → インポート対応（形式による）
✓ 添付ファイル → 手動アップロード（ドラッグ&ドロップ）
✓ 移行中も既存ツールと並行運用OK
✓ つまずいたらサポートチャットで即相談
```

## 11. FAQ（セキュリティ・権限・監査の情報を追加）

### 既存FAQ
- 無料トライアル期間中に課金されますか？ → いいえ。
- 導入にどのくらいの時間がかかりますか？ → 最短5分。
- いつでも解約できますか？ → はい。

### 新規追加FAQ

```
Q: Backlogから移行できますか？
A: はい。BacklogのCSVエクスポートをインポートするだけでタスク・マイルストーンが
   自動登録されます。コメント履歴もインポート対応。移行ガイドとサポートも用意しています。

Q: セキュリティは大丈夫ですか？
A: 通信はTLS 1.3、保存データはAES-256で暗号化。監査ログで全操作を記録しています。
   ポータルURLは個別発行・期限付き。IPアドレス制限も設定可能です（Businessプラン）。

Q: 権限管理はどうなっていますか？
A: ロール別の権限設定に対応しています。管理者・編集者・閲覧者・クライアント・ベンダー
   の5ロール。各ロールで見える情報と操作できる範囲が異なります。

Q: 承認フローは段階的に設定できますか？
A: はい。一次承認・最終承認の多段階フローに対応しています。差し戻し時のコメント記録、
   承認履歴のCSVエクスポートも可能です。

Q: 代理店モードは別料金ですか？
A: Businessプランに含まれています。追加料金はかかりません。

Q: クライアントのポータル利用に費用はかかりますか？
A: いいえ。ポータルは完全無料。招待するクライアント数に制限もありません。

Q: 監査対応のためのエクスポートはできますか？
A: はい。操作履歴・承認履歴・タスク一覧をCSV/PDFでエクスポート可能です。
   監査法人への提出用フォーマットにも対応しています。

Q: SLAはありますか？
A: Businessプラン以上で稼働率99.9%のSLAを提供しています。
   優先サポート・専任担当者による導入支援も含まれます。
```
