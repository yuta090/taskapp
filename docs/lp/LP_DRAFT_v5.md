## AgentPMの正確な差別化ポイント（調査反映）

BacklogやJiraもAI/MCPに対応し始めた今、AgentPMの差別化は「AI対応」単体ではなく:

1. **AI/MCP + クライアントポータル + ボール管理の統合** — 他ツールは個別対応。AgentPMは全て連動
2. **アカウント不要のクライアントポータル** — Backlogはゲストもアカウント作成必須
3. **ボール管理（誰待ちの可視化）** — どのツールにもない独自概念
4. **代理店モード（原価/マージン/売値）** — PMツールで唯一
5. **GitHubマージ → タスク完了 → クライアント承認依頼の自動連鎖** — 個別には可能でも、一気通貫は他にない
6. **日本語ネイティブ + モダンUI** — Backlogは古い、Linearは英語、Jiraは複雑
