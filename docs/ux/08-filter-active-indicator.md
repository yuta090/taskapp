# UX-08: フィルタ適用中の視覚表示

## 概要

タスクフィルター適用時に、どのフィルターが有効か一目で分かるチップ表示を追加する。

## 問題

- フィルタボタンにカウントバッジはあるが、何がフィルターされているか不明
- フィルターメニューを開かないと適用内容を確認できない

## 対応

### ActiveFilterChips コンポーネント

- フィルターカテゴリごとに1つのチップを表示
- チップ内容: `カテゴリ名: 選択値, 選択値` 形式
- 各チップにXボタンで個別解除が可能
- スタイル: `bg-blue-50 text-blue-700 border-blue-200` (WCAG AA準拠)

### 対象カテゴリ

| カテゴリ | チップ表示例 |
|---------|------------|
| ステータス | ステータス: 進行中, 完了 |
| ボール | ボール: 外部 |
| タイプ | タイプ: 仕様 |
| 担当者 | 担当者: 田中, 未割り当て |
| マイルストーン | MS: v1.0 |
| 優先度 | 優先度: 緊急, 高 |
| 期限 | 期限: 今週まで |
| 仕様状態 | 仕様: 決定 |
